# kael_ds

这个几个月一直在学Linux C，已经快要学完了，我以前是用Python的，里面内置了很多数据结构，而C语言内置是没有的，所以，就只能自己写一个，正好，数据结构的知识一直没学好，而且C语言也不熟，正好就来给自己写一个数据结构的库给自己用，写这个库对我自己有以下7个好处：
1. 熟悉下c的代码编写，
2. 学会C工程的开发，
3. 给自己准备一个c数据结构的库
4. 以后不用再编写这些代码
5. 学习下c语言单元测试的写法
6. 学习下c工程makefile的写法
7. 巩固数据结构的知识
8. 然后我将这个库的名字命名为：kael_ds。kael，为我的网名，ds为数据结构的缩写。

编写这个项目说明，希望这是一个很好的开始。
